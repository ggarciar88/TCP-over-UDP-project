
# TCP over UDP - ELITE Version

Simulación avanzada estilo TCP Reno sobre UDP.

## Características

- Three-way handshake
- Retransmisión automática
- Simulación de pérdida de paquetes (20%)
- Slow Start
- Congestion Avoidance
- ssthresh dinámico
- Registro de evolución de cwnd
- Script de visualización

## Ejecutar

1️⃣ Terminal 1:
python server.py

2️⃣ Terminal 2:
python client.py

3️⃣ Visualizar gráfica:
python plot_cwnd.py

Puerto: 9002

Proyecto ideal para demostrar conocimientos en:
- Networking avanzado
- Control de congestión TCP
- Sistemas distribuidos
